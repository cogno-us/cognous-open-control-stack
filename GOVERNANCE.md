# Governance

- This repository is initially maintained by Cognous.
- The repo is a public hub for the Cognous Open Control Stack.
- It does not define ODES.
- It does not define legal or compliance obligations.
- Maintainers evaluate changes based on clarity, public safety, vendor-neutrality, compatibility with the four-layer stack, and avoidance of overclaiming.
- Future governance may evolve if outside contributors or adopters participate.
