# Contributing

Contributions are welcome as issues or pull requests.

Please keep contributions aligned with the purpose of this repository:
- keep language vendor-neutral and public-safe;
- do not introduce proprietary implementation details;
- do not introduce private or internal terminology;
- preserve the ODES distinction;
- avoid overclaiming compliance, security, or certification;
- prefer small, reviewable changes.

When contributing:
- link related issues or context where helpful;
- keep documentation focused on the public hub role of this repository;
- avoid adding implementation code, package metadata, or monorepo-style structure;
- verify Markdown renders cleanly on GitHub before submitting.
