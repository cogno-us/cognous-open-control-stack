# Customer-Service Artifact Map

| Layer | Artifact | Example record type | Component repo |
|---|---|---|---|
| Declare | Agent Action Manifest | manifest JSON | cognous-agent-action-manifest |
| Control | Agent Control Plane | run record / policy decision records | cognous-agent-control-plane |
| Replay | Agent Replay Bundle | replay bundle JSON | cognous-agent-replay-bundle |
| Evidence | Agent Governance Evidence Pack | evidence pack JSON / rendered report | cognous-agent-governance-evidence-pack |

This map is intended to show how a single customer-service workflow can move through the full artifact chain. It is not a conformance test and does not define ODES conformance.
