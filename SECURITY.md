# Security Policy

This repository is a documentation and public hub repository. It does not contain runtime code, credentials, production infrastructure, or executable security controls.

## Reporting a vulnerability

If you believe you have found a security issue in this repository or one of the related component repositories, please report it by contacting:

andredelima@cogno.us

Do not include sensitive information in public GitHub issues.

## Scope

In scope:
- accidental disclosure of sensitive information;
- broken links to malicious or incorrect locations;
- documentation errors that could create unsafe implementation assumptions.

Out of scope:
- requests for security guarantees;
- compliance certification claims;
- production deployment review;
- vulnerabilities in downstream systems not maintained by Cognous.
